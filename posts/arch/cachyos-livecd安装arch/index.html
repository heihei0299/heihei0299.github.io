准备工作#

下载 CachyOS ISO#

从 CachyOS 官网下载 ISO 镜像，放入 Ventoy U 盘。

配置 Live 环境#

配置国内镜像源#

sudo bash -c 'echo "Server = https://mirrors.ustc.edu.cn/cachyos/repo/\$arch/\$repo" > /etc/pacman.d/cachyos-mirrorlist && echo "Server = https://mirrors.ustc.edu.cn/cachyos/repo/\$arch_v3/\$repo" > /etc/pacman.d/cachyos-v3-mirrorlist && echo "Server = https://mirrors.ustc.edu.cn/cachyos/repo/\$arch_v4/\$repo" > /etc/pacman.d/cachyos-v4-mirrorlist && echo "Server = https://mirrors.ustc.edu.cn/archlinux/\$repo/os/\$arch" > /etc/pacman.d/mirrorlist'

再次执行以下命令，防止安装过程中镜像源被替换：

sudo sed -i 's/^/#/' /etc/calamares/scripts/update-mirrorlist

分区与挂载#

使用 GParted 分区#

启动 GParted（CachyOS 自带），按以下方案分区：

挂载点大小文件系统标记
/efi2GFAT32boot, esp
/300Gext4
/home400Gext4

挂载分区#

使用下面的命令查看，硬盘分区情况。

sudo fdisk -l

  1. 将根分区挂载到 /mnt

  2. 创建 EFI 和 home 挂载点

    mkdir -p /mnt/efi /mnt/home

  3. 挂载 EFI 分区和 home 分区

    mount /dev/efi_partition /mnt/efi
mount /dev/home_partition /mnt/home

安装系统#

安装基础包#

pacstrap -K /mnt base linux linux-firmware btrfs-progs networkmanager vim sudo amd-ucode

-K 初始化密钥；base 基本包；linux 是内核，可以更换；linux-firmware 是固件；btrfs-progs 是 Btrfs 文件系统的管理工具；networkmanager 是联网用的；vim 是文本编辑器；sudo 和权限管理有关；amd-ucode 是微码，Intel 用户安装 intel-ucode。

pacstrap 是把软件安装到指定的根目录下。

如果你使用的是 marvell 的无线网卡，这里要额外安装 linux-firmware-marvell，否则进系统找不到网卡。

Vim 是以键盘操作为核心理念的文本编辑器。

  • i — 进入编辑模式
  • esc — 回到普通模式
  • u — 撤销
  • / — 搜索模式，n 下一个，Shift+n 上一个
  • :w — 写入
  • :q — 退出
  • :wq — 保存并退出
  • ! — 强制执行

不习惯的话可以安装 nano，记住 Ctrl+F 搜索、Ctrl+S 保存和 Ctrl+X 退出即可。也可以选择 neovim，Vim 的加强版，后面所有 vim 命令都改成 nvim。

  1. 创建 Swap 文件（64g 按实际需求调整）

    btrfs filesystem mkswapfile --size 64g --uuid clear /mnt/swap/swapfile

  2. 启用 Swap

    swapon /mnt/swap/swapfile

生成 fstab#

系统根据 fstab 自动挂载分区。

genfstab -U /mnt > /mnt/etc/fstab

genfstab — 生成文件系统表；-U 用 UUID 指定分区；> 输出覆盖写入，>> 追加写入。

更换根目录（Chroot）#

进入新安装的系统。

arch-chroot /mnt

此时根目录从 Live 环境变成了 /mnt，提示符会变化。

系统配置#

时区与时间#

timedatectl set-timezone Asia/Shanghai

同步硬件时间：

hwclock --systohc

也可以手动创建链接：

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

ln — link；-s 软链接；-f 强制执行。

本地化设置#

  1. 编辑配置文件

    vim /etc/locale.gen

    去掉 en_US.UTF-8 UTF-8 和 zh_CN.UTF-8 UTF-8 前面的 # 注释。

  2. 生成本地化配置

    locale-gen

  3. 设置系统语言

    vim /etc/locale.conf

    写入：

    LANG=en_US.UTF-8

    不要写 zh_CN.UTF-8，会导致 TTY 中文字符变成豆腐块。桌面装好后在用户空间再改语言。

主机名#

vim /etc/hostname

写入自己喜欢的主机名，比如 archlinux。

Root 密码#

passwd

输入过程不显示，直接输入回车即可。

引导与启动项#

安装 GRUB#

  1. 安装必要的软件包

    pacman -S grub efibootmgr os-prober exfat-utils

    efibootmgr 管理 UEFI 启动项；os-prober 和 exfat-utils 搜索 Win11（不配置双系统可跳过）。

  2. 安装 GRUB

    grub-install --target=x86_64-efi --efi-directory=/efi --boot-directory=/efi --bootloader-id=ARCH
    参数说明
    --target指定架构
    --efi-directory指定 ESP 位置
    --boot-directoryGRUB 安装目录
    --bootloader-id启动项名字，可自定义

    移动设备或主板只支持默认 EFI 路径时加 --removable。

  3. 编辑 GRUB 配置源文件

    vim /etc/default/grub
    • 启动项记忆 — GRUB_DEFAULT=0 改为 =saved，取消 GRUB_SAVEDEFAULT=true 的注释
    • 显示开机日志 — 去掉 quiet，设置 loglevel=5
    • 禁用 watchdog — 添加 nowatchdog 和 modprobe.blacklist=sp5100_tco（Intel CPU 换成 iTCO_wdt）
    • 搜索其他系统 — 取消 GRUB_DISABLE_OS_PROBER=false 的注释

  4. 在 GRUB 默认位置创建链接

    ln -sf /efi/grub /boot/grub

  5. 生成 GRUB 配置文件

    grub-mkconfig -o /boot/grub/grub.cfg

ZRAM#

zswap - ArchWiki | zram - ArchWiki

ZRAM 将内存的部分空间用作交换空间。如果没有配置 Swap，请一定配置 ZRAM。

  1. 安装 zram-generator

    pacman -S zram-generator

  2. 编辑配置文件

    vim /etc/systemd/zram-generator.conf

    [zram0]
zram-size = ram
compression-algorithm = zstd

  3. 禁用 zswap

    zswap 与 ZRAM 功能重复，禁用之。

    vim /etc/default/grub

    在 GRUB_CMDLINE_LINUX_DEFAULT 中添加 zswap.enabled=0。

  4. 重新生成 GRUB 配置文件

    grub-mkconfig -o /boot/grub/grub.cfg

启用网络服务#

systemctl enable NetworkManager

NetworkManager 默认使用 wpa_supplicant，可更换为更现代的 iwd。

部分设备更换后可能无法正常联网。

pacman -S iwd
mkdir -p /etc/NetworkManager/conf.d
vim /etc/NetworkManager/conf.d/iwd.conf

写入：

[device]
wifi.backend=iwd

完成安装#

退出并重启#

exit
reboot

此时自动取消所有挂载。

选择 UEFI 启动项#

通常默认就是刚安装的 Arch。如果未出现启动项，参考 ArchWiki GRUB。

登录并联网#

登录 root 账户（密码刚才已设置）。

  1. 验证网络

    ip a
ping bilibili.com

  2. 连接 WiFi

    nmtui
    • 选择 Activate a connection
    • 选择自己的 WiFi 连接
    • Esc 退出，Ctrl+L 清屏
