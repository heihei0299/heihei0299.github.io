纯 IPv6 机器安装 Nezha Agent#

1. 下载并安装 Agent#

wget https://github.com/nezhahq/agent/releases/download/v0.16.2/nezha-agent_linux_amd64.zip
mv nezha-agent /usr/local/bin/
chmod 744 /usr/local/bin/nezha-agent

2. 测试启动#

nezha-agent -s <Dashboard服务器地址>:<端口> -p <连接密码>

如果面板中显示机器在线，说明测试成功。

3. 配置 systemd 服务#

在 /usr/lib/systemd/system 下创建 nezha-agent.service：

vim /usr/lib/systemd/system/nezha-agent.service

内容如下：

[Unit]
Description=Nezha Agent
After=syslog.target

[Service]
Type=simple
User=root
Group=root
ExecStart=/usr/local/bin/nezha-agent -s <Dashboard服务器地址>:<端口> -p <连接密码>
Restart=always

[Install]
WantedBy=multi-user.target

4. 启动服务#

systemctl daemon-reload
systemctl enable nezha-agent
systemctl start nezha-agent

5. 查看状态#

systemctl status nezha-agent
