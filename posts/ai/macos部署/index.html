macos部署#

硬件要求#

硬件要求：

  • CPU：M系列芯片或Intel i5以上
  • 内存：8GB以上（推荐16GB）
  • 硬盘：10GB以上空闲空间

系统版本：

  • macOS 12 Monterey 或更高版本
  • 推荐 macOS 14 Sonoma 或 macOS 15 Sequoia

环境准备#

安装homebrew#

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

常见错误点击这里查看

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装的时候选择阿里源

验证homebrew

brew -v

安装 Nodejs#

安装nvm

brew install nvm

设置环境变量

vim ~/.zprofile

将以下内容添加到.zprofile 文件最后面

export NVM_DIR="$HOME/.nvm"
[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"  # 加载 NVM
[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"

按 ESC + :wq 保存并退出编辑器。

加载环境变量到环境中

source ~/.zprofile

接下来使用nvm安装nodejs

nvm install 22

配置NVM节点镜像源 (加速Node下载)#

安装好NVM后，将节点镜像地址设置为国内源。

临时设置（当前终端有效）：

bash

export NVM_NODEJS_ORG_MIRROR=https://npmmirror.com/mirrors/node  # 淘宝源 [6]
# 或
export NVM_NODEJS_ORG_MIRROR=https://mirrors.ustc.edu.cn/node/    # 中科大源 [11]

永久设置： 将上述 export 命令添加到你的 Shell 配置文件（.bashrc, .zshrc 或 .bash_profile）中。

  1. NVM for Windows 设置

打开 nvm 安装目录下的 settings.txt 文件，添加以下行：

text

node_mirror: https://npmmirror.com/mirrors/node/
npm_mirror: https://npmmirror.com/mirrors/npm/

验证nodejs的版本#


# Verify the Node.js version:
node -v # Should print "v22.14.0".

# Verify npm version:
npm -v # Should print "11.9.0".

切换npm的到国内淘宝镜像

npm config set registry https://registry.npmmirror.com
npm config get registry

安装git#

brew install git

验证git安装

git -v

安装openclaw#

npm i -g openclaw

验证安装

openclaw --version

如果显示版本号就是安装成功

运行以下命令初始化openclaw

openclaw onboard
