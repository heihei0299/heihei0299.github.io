安装mcp并使用mcp#

准备工作：安装 mcporter CLI#

在配置 MCP 服务器之前，先安装一个神器——mcporter CLI。

mcporter 是一个专门用于管理 MCP 服务器的命令行工具，可以快速配置、测试、管理 MCP 服务器。

安装很简单：

npm install -g mcporter

安装完成后，会自动创建配置文件和技能框架，这为后续配置打好了基础。在wabui界面启用 mcporter skills

或者直接安装mcporter 技能

clawhb install   mcporter
